About the Europeana - Random image Explorer


See live demo here.

This program is a somewhat polished up version, of a prototype developed at the Europeana Hackathon, it was one of the price winning prototypes. It used to contain a black cat, but I changed it to use the Europeana picture API.

If you move the mouse over one of the images, the search box gets populated with a new query based upon the metadata from that object, if you click on an image, the query runs.

It depends on a lot of python stuff, and is mainly built upon webpy, mako, jquery. The client does a request containing a query, the query is resolved via the Europeana API. The client receives a page containing ajax requests to the full objects (/get/). The program fetches the full object descriptions and returns them in json format.

Code was written by Willem Jan Faber, ( Koninklijke Bibliotheek - National Library of the Netherlands ).

See the RiE source files, or tarball. For an API key contact Europeana.
